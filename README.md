# RobustSR

## Reproducibility
Please follow steps below to reproduce the results of JI-DCL reported in our paper on the four datasets.

1. Run 
```python
  python main.py
```
2. Input
```python
  RobustSR_lastfm
```

```python

  RobustSR_Douban_book
```
